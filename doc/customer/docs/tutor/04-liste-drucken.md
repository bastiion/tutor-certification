# Liste drucken

## Ziel

Hier werden klassische Druckbescheinigungen aus einer Namensliste
(eine Zeile pro Teilnehmer:in) erstellt — ohne kryptographische Signatur.

## Schritt-für-Schritt

1. Zu `/tutor/print` navigieren oder auf der
   Startseite auf **Drucken** klicken.

2. Die Felder ausfüllen:

    - **Titel** — z. B. „Teilnahmebescheinigung"
    - **Datum** — z. B. „14. April 2026"
    - **Name der Einrichtung** — die ausstellende Organisation
    - **Beschreibung / Fließtext** — optionaler Freitext
    - **Logo** — optionales Bild
    - **Namen** — einer pro Zeile

3. Auf **Generieren** klicken, um die Vorschau zu erzeugen.

4. Auf **Drucken** klicken, um den Druckdialog zu öffnen.

!!! warning "Hinweis"
    Diese Druckbescheinigungen sind nicht kryptographisch signiert.
    Für verifizierbare Bescheinigungen ist der reguläre
    Ablauf über [Sitzung erstellen](03-sitzung-erstellen.md) zu verwenden.
