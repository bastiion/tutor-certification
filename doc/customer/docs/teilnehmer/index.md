# Teilnehmer:in

Teilnehmende erhalten von ihrer Kursleitung einen
Einschreibe-Link. Über diesen Link wird die Bescheinigung
beantragt und heruntergeladen.

## Ablauf

1. [Link öffnen](01-link-oeffnen.md) — Einschreibe-Link im Browser aufrufen
2. [Bescheinigung erhalten](02-bescheinigung-erhalten.md) — Name eingeben und absenden
3. [Bescheinigung sichern](03-bescheinigung-sichern.md) — JSON herunterladen oder drucken
